Async SoundManager Documentation

Description

The SoundManager script creates on the first call a SoundManager GameObject. This new GameObject holds all the AudioSource generated by this script.

Do not interact with these AudioSources outside of the SoundManager script. Otherwise, it can produce unexpected results.

You can use 4 different sound groups, OneTime, Effect, FX and Music. Each group has its properties, list here:

Group As Cache Can Loop DoNotDestroy Support FadeIn Support Special Feature Note
OneTime x x x x Can specify, on-call, a AudioMixerGroup "Fire and forget". Calling the same AudioClip creates a new AudioSource.
Effect ✔ x ✔ ✔ x Ideal for short sounds often called. Calling the same AudioClip restart the one already playing.
FX ✔ ✔ ✔ ✔ x Ideal for long-living background sounds. Calling the same AudioClip restart the one already playing.
Music ✔ ✔ x ✔ Starting a new AudioClip, crossfade the current with the new one during the transition. Only one music can play at a time. Calling the same AudioClip restart the one already playing.

Static Properties

Name Type Accessibility Description
AudioMixer AudioMixerGroup {get;set;} Optional - The default/fallback AudioMixer. If you had specified an AudioMixer here but none in EffectAudioMixer, FxAudioMixer or MusicAudioMixer, they fall back to this one.
EffectAudioMixer AudioMixerGroup {get;set;} Optional - The Effect group AudioMixer. You can specify an AudioMixer here to apply it on the Effect group.
FxAudioMixer AudioMixerGroup {get;set;} Optional - The FX group AudioMixer. You can specify an AudioMixer here to apply mixer effects on the FX group.
MusicAudioMixer AudioMixerGroup {get;set;} Optional - The Music group AudioMixer. You can specify an AudioMixer here to apply mixer effects on the Music group.
MainVolume float {get;set;} Set the main volume. It only affects the SoundManager managed AudioSources. It appends to the value specified in MainEffectVolume, MainFxVolume and MainMusicVolume.
MainEffectVolume float {get;set;} Set the Effect group's main volume.
MainFxVolume float {get;set;} Set the FX group's main volume.
MainMusicVolume float {get;set;} Set the Music group's main volume.

Static Methods

  • Prepare - Add to the SoundManager cache a SoundManagerAudio object to preload it and accelerate the first call.
  • Play - Play an AudioClip. If you don't set the option doNotDestroy to true, the AudioSource component deletes itself after the AudioClip has done playing.
  • Pause - Pause the AudioClip. The AudioSource still exists until the method Play on the SoundManagerAudio is called again.
  • Stop - Stop the AudioClip. The AudioSource component deletes itself after the specified Fadeout time.
  • Clear - Clear the cache and the AudioSource element from the SoundManager GameObject.
  • Get - Return the SoundManagerAudio object associated with the specified Id or AudioClip from the cache. If the search doesn't return a result, it throws an error.
  • Find - Return the SoundManagerAudio object associated with the specified Id or AudioClip from the cache. If the search doesn't return a result, it returns null instead.

Each method has the version allowing accessing the internal cache using Id, SoundManagerAudio or AudioClip.
