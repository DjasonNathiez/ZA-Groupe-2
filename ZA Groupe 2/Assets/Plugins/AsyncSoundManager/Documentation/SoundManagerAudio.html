SoundManager.SoundManagerAudio Documentation

Description

This class is used by the SoundManager script to wrap the Unity AudioClip class and add more advanced functionalities.

Properties

Name Type Accessibility Description
AudioClip AudioClip {get;} The AudioClip that will be play when using the SoundManagerAudio object.
DoNotDestroy bool {get;} If the SoundManagerAudio object will be automatically deleted after done playing.
Duration float {get;} The duration of the AudioClip.
IsPLaying bool {get;} If the SoundManagerAudio is currently playing.
