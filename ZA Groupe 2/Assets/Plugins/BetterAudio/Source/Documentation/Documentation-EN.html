Delay
On Start
On Play
On Update
On End
On Collider
On Event
When triggered the Delay send a command either Play, Stop or Toggle after the specified delay.
  • Play: if the Audio Source is paused or stopped it will play.
  • Stop: if the Audio Source is playing it will stop it.
  • Toggle: if the Audio Source is playing it will stop, however it will play

The Fade behaviour will set the volume at the Initial amount. Then the amount will go to the Target amount taking the time specified in the Fade Time property.

When used in Update behaviour, you must specify a time at wich it will be triggered.

Follow, will translate the source's gameobject to another specified transform.
If Keep Offset is checked, the behaviour will remenber the distance between the target and the source when First triggered and keep it same for all future triggers.
Loop

On Update

On Collider
On Event
When the clip is finished, the Loop behaviour will rewind the play time to 0 and play it again. The Loop count property will tell how many times the behaviour will trigger. If Auto Destroy is active, the source's gameobject will be destroy once all loops done and after the property delay.
