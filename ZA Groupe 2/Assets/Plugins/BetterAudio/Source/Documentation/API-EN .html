BetterAudio Source API

BetterAudioEvent.SendEvent (string Event Name, out GameObject|[] Target|Targets, object argument)
Events are only available for the Plus version

SendEvent will trigger all behaviours in "OnEvent" categorie with matching name for all BetterAudio Sources in scene.
The function can output the last target or all triggered GameObject.
Argument are not usable for now.
